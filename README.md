# SVG Drawing Editor

A fully interactive vector graphics editor that runs directly in the browser,
built with vanilla JavaScript and the SVG DOM API — no external libraries.

## Live Demo
🔗 https://andrei-m-popescu.github.io/svg-drawing-editor

## Features
- Draw lines, rectangles, ellipses, and freehand paths
- Stroke and fill color controls
- Element selection and drag-and-drop repositioning
- Path editing with control point handles
- Undo stack, Delete, Save SVG, Export PNG

## How to Run
No installation needed — just open `index.html` in any modern browser.
Or use the live demo link above.

## Tech Stack
- Vanilla JavaScript (ES6)
- HTML5 SVG DOM API
- Canvas API (PNG export)
- CSS3
